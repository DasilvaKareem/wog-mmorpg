## Progression Flow

Quests must be completed in order. Each quest unlocks the next in the chain. Completing the final quest in a zone unlocks the first quest in the next zone.

```
HUMAN MEADOW (Guard Captain Marcus)
├─ 1. Rat Extermination (L1) — Kill 3 Giant Rats — 25g, 50 XP
│   └─ 2. Wolf Hunter (L2) — Kill 5 Hungry Wolves — 50g, 100 XP
│       └─ 3. Boar Bounty (L2) — Kill 4 Wild Boars — 60g, 120 XP
│           └─ 4. Goblin Menace (L3) — Kill 3 Goblin Raiders — 75g, 150 XP
│               └─ 5. Slime Cleanup (L3) — Kill 2 Mire Slimes — 60g, 120 XP
│                   └─ 6. Bandit Problem (L4) — Kill 3 Bandit Scouts — 100g, 200 XP
│                       └─ 7. The Alpha Threat (L5) — Kill 1 Diseased Wolf — 155g, 310 XP
│                           ↓ UNLOCKS WILD MEADOW
WILD MEADOW (Ranger Thornwood)
├─ 8. Bear Necessities (L6)
│   └─ 9. Arachnophobia (L7)
│       └─ 10. Outlaw Justice (L8)
│           └─ 11. Nature's Corruption (L9)
│               └─ 12. The Pack Leader (L10)
│                   └─ 13. Wilderness Survival (Challenge)
│                       ↓ UNLOCKS DARK FOREST
DARK FOREST (Priestess Selene)
├─ 14. Shadows in the Dark (L11)
│   └─ 15. Cult Cleansing (L12)
│       └─ 16. Undead Purge (L13)
│           └─ 17. Troll Slayer (L14)
│               └─ 18. Golem Breaker (L15)
│                   └─ 19. The Necromancer's End (L16 BOSS)
│                       └─ 20. Master of the Dark Forest (ULTIMATE)
```

## Gate Quests

- **The Alpha Threat** (Quest 7) — clears human-meadow, unlocks wild-meadow
- **Wilderness Survival** (Quest 13) — clears wild-meadow, unlocks dark-forest
